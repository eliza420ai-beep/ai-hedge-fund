# Anthropic marketing workflow (reference)

One-person marketing at scale: Austin Lau (non-technical growth lead) ran paid search, paid social, email, and SEO solo using Claude Code, agents, and MCP.

## Workflow

1. **Export ad CSVs into Claude Code** — structured input, no manual copy-paste.
2. **AI flags underperforming ads** — analysis and prioritisation in one place.
3. **Agents generate new headlines + descriptions** — iteration at volume.
4. **Figma auto-swaps copy across 100 ad templates** — bulk creative updates.
5. **MCP server pulls live Meta data** — fresh performance data without manual exports.

## Results

- Ad creation: **2 hours → 15 minutes**
- Total marketing output: **~10×**
- Conversion rates: **41% above industry average**

## Why it works (Claude Architect lens)

- **Structured in/out:** CSVs and tool outputs, not prose handoffs.
- **Right tools, right jobs:** Claude Code for analysis + generation; MCP for live data; Figma automation for template updates.
- **Human as orchestrator:** One person sets goals and quality; agents and tools handle volume and iteration.

Use this pattern when you want one person to operate at “whole team” scale with clear pipelines and tool boundaries.
